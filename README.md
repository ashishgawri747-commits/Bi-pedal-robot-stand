# Bi-pedal-robot-stand
I made a 15 cm tall stand to hold my Bi-pedal, it will help me test the droid without it falling down, i started testing the leg movement and it was hard to do on a cardboard box that didn't even grip it. Using it is pretty easy, you just pick up your droid and put it on the stand and there you go.
